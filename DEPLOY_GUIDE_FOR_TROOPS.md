#  旅團部署指南

> 10 分鐘完成。從前端下載 Code.gs，建立 Google Sheet，部署 Apps Script。

---

## 快速開始

**訪問系統前端頁面：**
```
https://vsbadge.vercel.app/
```

點擊頂部的 **「 新旅團部署」** 按鈕，即可看到完整部署教學和 Code.gs 下載按鈕。

---

## 部署步驟

### 第 1 步：下載後端程式碼

從系統前端頁面下載 `Code.gs` 檔案：
- 訪問 `https://vsbadge.vercel.app/`
- 點擊 **「 新旅團部署」**
- 點擊 **「下載 Code.gs」** 按鈕

### 第 2 步：建立 Google Sheet

1. 開啟 [Google Sheets](https://sheets.google.com)
2. 建立新試算表
3. 命名為「深資童軍進度追蹤 - 您的旅團名稱」
4. 複製網址中的 **Sheet ID**：
   ```
   https://docs.google.com/spreadsheets/d/【這段就是 Sheet ID】/edit
   ```

### 第 3 步：部署 Apps Script

1. 在 Sheet 中，點擊 **「擴充功能」→「Apps Script」**
2. 刪除所有預設程式碼
3. 開啟下載的 `Code.gs` 檔案，**複製全部內容**
4. **貼上**到 Apps Script 編輯器
5. 修改第一行的 `SHEET_ID`：
   ```javascript
   const SHEET_ID = '您的Sheet ID';  // 替換為第 2 步複製的 ID
   ```
6. 點擊 💾 **儲存**
7. 在函數選單選擇 `initializeSheets`，點擊 ▶ **執行**
8. 首次執行會要求授權：
   - 選擇您的 Google 帳號
   - 點擊「進階」→「前往」→「允許」
9. **系統會自動完成以下設定：**
   - 建立「進度追蹤」工作表
   - 建立「成員名單」工作表
   - 建立「Users」工作表（含超管帳號）
   - 建立「Applications」工作表（申請紀錄）
   - 建立「Tokens」工作表
   - 建立「SystemConfig」工作表（登入模式設定）
   - 生成 API Key
   - 建立超管帳號（YMIS: `sheep`, 密碼: `0728`，隱藏，開發測試用）

10. **複製顯示的 API Key 和超管帳號資訊**

### 第 4 步：部署為網頁應用程式

1. 點擊 **「部署」→「新增部署」**
2. 點擊  齒輪 → 選擇 **「網頁應用程式」**
3. 設定：
   - 描述：深資童軍進度追蹤
   - 執行身分：我
   - 存取權：任何人
4. 點擊 **「部署」**
5. 複製產生的 **網頁應用程式網址**（以 `/exec` 結尾）

### 第 5 步：提交給系統管理員

把以下資訊交給系統管理員：

| 資訊 | 說明 |
|------|------|
| 旅團編號 | 例：0082 |
| Apps Script URL | 第 4 步複製的網址 |
| API Key | 第 3 步顯示的隨機字串 |

管理員會將您的旅團加入系統，之後您和成員就能直接使用。

---

### 第 6 步：（選用）接入主系統

如果您的旅團有使用 scoutsystem-2.0 主系統，可以把此系統接入：

1. 把上述 3 項資訊交給主系統管理員
2. 管理員會在主系統的「元件設定」中加入您的旅團
3. 設定完成後，從主系統 Dashboard 的元件卡片就能直接進入
4. 身份會自動帶入，不需要再選擇旅團和身份

**提示：** 獨立使用和接入主系統可以同時使用，不衝突。領袖可以從主系統進入（身份自動帶入），成員可以直接 bookmark 前端 URL。

##  首次登入

### 超管帳號（開發測試用，隱藏）

- **YMIS**: `sheep`
- **密碼**: `0728`
- **角色**: 超級管理員（隱藏，不顯示在 UI）
- **用途**: 系統開發和測試

**注意：** 超管帳號不會顯示在用戶列表中，只有知道帳號密碼的人才能登入。

### 建立管理員帳號（旅團第 1 個帳號）

管理員帳號需要透過申請流程建立：

1. 打開系統前端頁面
2. 選擇旅團
3. 點擊「還沒有帳號？申請加入」
4. 填寫：
   - YMIS：您的 YMIS（10 位數字）
   - 姓名：您的姓名
   - 申請角色：選擇「管理員」
5. 提交申請
6. 用超管帳號登入（sheep / 0728）
7. 進入「審批申請」頁面
8. 批准您的申請
9. 登出，用新帳號登入
   - **預設密碼**: `changeme`
   - **首次登入後請立即更改密碼**

##  用戶申請和審批流程

### 用戶申請

1. 打開系統前端頁面
2. 選擇旅團
3. 點擊「還沒有帳號？申請加入」
4. 填寫：
   - YMIS：10 位數字
   - 姓名：真實姓名
   - 申請角色：團員 或 主席/執行委員
5. 提交申請
6. 等待管理員審批

### 管理員審批

1. 用管理員/團長/支部領袖帳號登入
2. 點擊「 審批申請」標籤
3. 查看待審批申請列表
4. 點擊「批准」或「拒絕」
5. 批准後，系統會自動建立用戶帳號
   - **預設密碼**: `changeme`
   - 用戶可以用 YMIS 和預設密碼登入

### 權限管理

管理員/團長/支部領袖可以管理用戶權限：

1. 點擊「👥 用戶管理」標籤
2. 查看所有用戶列表
3. 可以調整：
   - **角色**: 團員 → 執委 → 支部領袖 → 團長 → 管理員
   - **勾選權限**: 授權/收回勾選進度的權限

**權限層級：**
- 管理員 > 團長 > 支部領袖 > 執委 > 團員
- 上級可以管理下級的權限
- 不能給自己或上級設定權限

##  切換登入模式

### 方法 1：直接修改 Sheet

1. 打開 Google Sheet
2. 找到 `SystemConfig` 工作表
3. 修改 `login_mode` 的值：
   - `main_system` — 使用主系統登入（信任 URL 參數）
   - `standalone` — 使用獨立系統登入（需要 YMIS + 密碼）
4. 保存

### 方法 2：超管/管理員設定頁面

1. 登入系統
2. 點擊「設定」→「系統設定」
3. 切換「登入模式」開關
4. 保存

## 🆘 常見問題

**Q: 授權失敗？**
A: 確保您使用建立 Sheet 的同一個 Google 帳號。

**Q: 顯示「無法連接後端」？**
A: 確認 Apps Script 已部署為「網頁應用程式」且存取權設為「任何人」。

**Q: 忘記 API Key？**
A: 在 Apps Script 編輯器中，選擇 `showApiKey` 函數，點擊 ▶ 執行，會再次顯示 API Key。

**Q: 其他裝置要用怎麼辦？**
A: 管理員設定好後，使用給您的連結就能在任何裝置上使用。登入狀態會儲存在瀏覽器，不需要每次登入。

**Q: 資料安全嗎？**
A: 資料存在您自己建立的 Google Sheet，只有您能存取。API Key 防止爬虫隨便存取。

**Q: 忘記密碼怎麼辦？**
A: 在登入頁面點擊「忘記密碼？」，輸入您的 YMIS 和 Email，系統會發送新密碼到您的 Email。

**Q: 如何新增成員？**
A: 成員可以自行申請，管理員審批即可。或者管理員直接在「用戶管理」頁面新增。

**Q: 如何授權團員勾選進度？**
A: 管理員/團長/支部領袖登入後，進入「用戶管理」，找到該團員，點擊「授權勾選」。

**Q: 超管帳號會顯示在 UI 上嗎？**
A: 不會。sheep / 0728 是隱藏帳號，只在後端使用，不會出現在用戶列表中。

**Q: 管理員預設密碼是什麼？**
A: `changeme`。首次登入後請立即更改密碼。
