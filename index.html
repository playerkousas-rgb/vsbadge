📋 新旅團部署指南 - 有主系統 / 無主系統 兩條路 (10分鐘)

🌟 先了解一下：主系統平台是什麼？

你而家用緊嘅 深資童軍進度追蹤 係一個 Tier 3 元件，可以獨立用，但其實屬於更龐大的 scoutsystem-2.0 主系統平台：

  • 📊 Dashboard 儀表板：全旅團人數、活動、財政一目了然
  • 👥 成員管理、✅集會考勤 QR Code、💰財務、📅活動報名
  • 🧩 插件市場：本系統是其中一個插件，可一鍵加入卡片
  • 🔐 單一登入：點卡片自動帶入身份，唔使再輸 YMIS/密碼

重點：有主系統 ≠ 一定要接上本系統，可自由選擇獨立用或接上，兩條路並存不衝突。藉此告訴用家有更大平台可用。

🗺️ 部署地圖 - 兩條路

共通：下載 Code.gs → 建 Sheet → 執行 initializeSheets → 部署 → 複製 URL+API Key
軌道A 無主系統/有但想獨立用： 將 URL+Key 交 vsbadge 管理員 → 加入 troops.json → 完成 → 用法：vsbadge.vercel.app 選旅團登入
軌道B 有主系統並想接上： 同樣交 vsbadge 管理員 + 額外交主系統管理員填入旅團設定→元件設定 backend+apikey → 用法：主系統 Dashboard 點卡片 → 自動帶入 from=portal&embed=1 → 直接用

私隱選擇

有主系統不等於一定要接，可自由選擇獨立用。若旅團重私隱，成員預設只看自己；若想互相鼓勵，團長可在用戶管理→系統設定開啟「允許成員互相查看進度」

兩條路可同時用，領袖從主系統入，成員 bookmark 獨立連結

📖 使用教學

1

選擇旅團

首頁點擊旅團卡片（例如 0082 第82旅）

2

登入

成員：YMIS 10位數字 + 密碼；領袖：Email + 密碼。首次登入預設密碼請洽管理員。

3

成員操作：申請完成

成員不可直接勾選，點擊「📝 申請完成」→選擇日期→填寫證據→提交。領袖會在「待批」頁看到。

4

領袖操作：勾選+日期

領袖可直接勾選，系統自動填今日日期，亦可手動改日期。所有勾選先存瀏覽器，底部顯示未保存數量，按「確認寫入」才批量寫入雲端，寫入時有進度提示。

5

其他功能

全團總覽：按段章篩選，避免卡死；按項目統計：百分比+人數；其他獎章已併入我的進度；表格列印：VS/DRAGON FORM官方格式自動填入後列印。

👥 角色功能教學（登入後）

👤 成員 member
  • 只能看自己進度
  • 不能直接剔，按📝申請
  • 待批查看自己申請
  • 其他獎章只讀自己
  • 表格列印官方格式
  • 資料庫看綱要及安全指引
🎖️ 執委 exec
  • 成員全部功能
  • 可勾選（需授權範圍）
  • 默認：會員章+活動段章(細項不用)+其他逐個勾
  • 可選成員查看紀錄
  • 全團批量剔（若授權）
  • 待批批准（若授權）
👨‍💼 領袖 leader
  • 全部功能
  • 我的進度選任何成員看紀錄
  • 全團一眼睇會員章+百分比
  • 考驗營批量一鍵幫全團剔
  • 離線暫存確認寫入
  • 待批批量批准
  • 用戶管理改成員為執委+⚙️權限設定
  • 領袖默認全部，團長/管理員可改低層級
  • 審批新帳號
