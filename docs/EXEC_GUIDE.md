# 🎖️ 執委使用教學 (Exec Committee)

> 你由成員晉升為執委，擁有部分領袖權限，可協助考核

## 你比成員多什麼？
- 成員的所有功能
- **可勾選進度**（需領袖授權可勾選範圍）
  - 默認可考核：**會員章 + 活動段章(細項不用勾，勾段章即可) + 其他獎章逐個勾**
  - 團長/管理員可在「用戶管理」→ 按你帳號旁「⚙️權限」設定你可考核哪些段章
- **可選成員查看**：在「我的進度」頂部可選其他成員，看他所有紀錄，方便策劃活動
- **全團批量**：若被授權，可在「👥全團總覽」使用「🚀考驗營批量完成」：
  1. 選日期
  2. 左選有去的成員（可搜尋過濾）
  3. 右選本次考驗的項目
  4. 一鍵標記完成 → 暫存瀏覽器 → 底部按確認寫入雲端

- **待批**：若被授權，可批准成員的「申請完成」

## 如何幫成員勾選？
1. 我進度頁選成員
2. 找到項目，直接勾選 + 改日期（自動填今日，可手改）
3. 所有勾選先存瀏覽器，底部顯示數量，按「確認寫入雲端」才批量寫入，有「正在寫入」提示

## 策劃活動
執委可按全團進度策劃活動：例如在全團總覽一眼睇邊個未完成會員章，針對性安排戶外活動。
