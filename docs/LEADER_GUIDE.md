# 👨‍💼 領袖使用教學 (Branch Leader / Group Leader / Admin / SHEEP)

> 你擁有全部權限，負責審批、授權、批量操作

## 核心功能

### 1. 我的進度 - 選成員看所有紀錄
- 頂部「檢視成員」下拉選任何人，查看其完整進度（包括其他獎章）
- 直接勾選 + 輸入日期（剔即填今日，可手改日期）
- 級聯自動完成：勾上層自動勾所有下層
- 離線暫存 → 底部 SaveBar 顯示數量 → 按確認才批量寫入雲端，有進度提示

### 2. 全團總覽 - 一眼睇 + 批量剔
- **卡片總覽**：每人完成度% + 四個獎章百分比，一眼睇邊個完成會員章（顯示會員章✓標籤）
- **表格矩陣**：按段章篩選，最多30欄防卡死，點✓可單獨切換
- **🚀考驗營批量完成**：例如剛去完考驗營
  - 選日期
  - 左選有去的成員（支援搜尋過濾，全選/取消全選）
  - 右選本次考驗的項目（支援全選）
  - 一鍵標記 → 暫存 → 確認寫入，唔使逐個跳轉
- **按成員/按項目統計**：子切換按鈕，一個分頁內完成（手機省闊度）

### 3. 待批 - 成員申請
- 成員「申請完成」後在此顯示，含日期、證據
- 可單個批准/拒絕，或批量選中批准
- 批准後自動加入待寫入清單

### 4. 表格列印 - 官方格式
- 自動填入姓名、YMIS、旅團、段章完成日期、其他獎章
- PT/19 深資童軍獎章、PT/20 榮譽章（龍章）官方格式，含個人摘要、完成項目、提名/推薦/批核簽署欄
- 可編輯後列印，雙面列印符合總會要求

### 5. 用戶管理 - 角色及細緻權限
- 將成員改為執委：選單改角色 + 授權勾選
- **⚙️權限按鈕**：
  - 領袖默認全部(*)
  - 成員默認無
  - 執委默認 L1會員章 + L3活動段章 + OTHER部分
  - 可按個別成員勾選可考核什麼：會員章、活動段章（細項不用）、其他獎章逐個勾
  - 高層級可設定低層級可勾選範圍，除非團長/管理員更改
- 授權/收回勾選權限

### 6. 系統設定 - 私隱開關
- **允許成員互相查看進度**：預設關（只看自己，重私隱），團長可開啟（互相鼓勵，執委可按進度策劃活動）
- 在用戶管理頁 → 系統設定 → 開關
- 成員是否可看其他人進度由此控制

### 7. 審批申請
- 審批新帳號申請（YMIS/Email）

### 8. 資料庫
- 第11版新綱要對照、訓練計劃、保護兒童 Safe from Harm 連結、佩戴指引、表格下載
