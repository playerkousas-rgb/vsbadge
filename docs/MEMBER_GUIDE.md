# 👤 成員使用教學 (Member)

> 你只需關心自己的進度，其他功能由領袖處理

## 登入後你會見到
- **📊 我的進度**：預設只顯示你自己的完成度
  - 綠色進度條 = 已完成比例
  - 點擊獎章展開段章，再展開項目
  - 每項右側有 ⓘ 查看考驗說明，不用翻綱要
  - **不能直接剔**：你只能按「📝 申請完成」

- **⭐ 其他獎章**：已併入「我的進度」最底虛線卡片，顯示急救章、AYP、拯溺等，你的紀錄在此

- **📝 待批**：查看你提交的申請是否已批，狀態：⏳待領袖確認 / ✅已批准

- **🖨️ 表格列印**：系統會自動根據你的進度填好 PT/19 (深資童軍獎章) 及 PT/20 (榮譽章/龍章) 所需日期，你可編輯地址、區、地域後直接列印，官方格式，記得雙面列印

- **📚 資料庫**：保護兒童 Safe from Harm 課程連結、訓練綱要、佩戴指引

- **❓ 教學**：此文件，按角色只顯示你需要的部分

## 常見操作
### 如何申請完成一個項目？
1. 在「我的進度」找到該項目
2. 按右側「📝 申請」
3. 選完成日期 + 填寫證據（例如：2026-05-01 考驗營相片3張）
4. 按提交，**先暫存瀏覽器**，領袖會在「待批」看到
5. 領袖批准後，底部會顯示已寫入，進度自動更新

### 為何我看不到其他成員？
預設為私隱，普通成員只能看自己。若旅團想互相鼓勵，團長可在「用戶管理」→「系統設定」開啟「允許成員互相查看進度」。

### 忘記密碼？
聯絡團長/支部領袖重設為 `changeme`，首次登入後請立即更改。
