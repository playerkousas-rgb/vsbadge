四級勾選架構

L1
進度獎章
會員章/肩章/深資童軍獎章/榮譽童軍獎章
L2
段章 / 金帶
活動策劃/社會服務/多元技能/戶外探險
L3
大項
露營/繩結/遠足/急救/個人興趣...
L4
小項
第 1 次出席/第 2 次出席/具體要求...

靈活勾選

領袖可以在任何層級勾選：

  • 勾 L4 小項 = 完成一個具體要求
  • 勾 L3 大項 = 完成大項（自動勾選所有 L4）
  • 勾 L2 段章 = 完成段章（自動勾選所有下層）
  • 勾 L1 獎章 = 完成獎章（自動勾選全部）

雙向級聯

勾選會自動聯動：

  • 勾上層 → 自動完成所有下層
  • 下層全完成 → 上層自動完成
  • 一天只能完成幾項？勾 L4 即可
  • 一次完成整個大項？直接勾 L3

👥 權限管理

根據角色提供不同功能：

  • 管理員：管理全旅團
  • 團長/支部領袖：管理所屬支部
  • 執委：可勾選進度（需授權）
  • 團員：只能查看自己的進度

📋 新旅團部署指南

✨ 只需 10 分鐘，3 步完成

您的旅團會擁有獨立的 Google Sheet 後端，資料完全由您掌控。

1

下載後端程式碼

⬇️ 下載 Code.gs

2

建立 Google Sheet 並部署

2.1 開啟 Google Sheets 建立新試算表
2.2 點擊「擴充功能」→「Apps Script」
2.3 刪除預設內容，貼上 Code.gs
2.4 儲存後執行 initializeSheets
2.5 部署為網頁應用程式，存取權設為「任何人」
2.6 複製部署後網址

3

提交給系統管理員

  • 旅團編號（例：0082）
  • Apps Script 網址
  • API Key

📧 完成部署後，請通知管理員

請將您的 旅團編號、Script URL 和 API Key 發送給系統管理員進行登記。
