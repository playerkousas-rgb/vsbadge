靈活勾選

領袖可以在任何層級勾選：

  • 勾 L4 小項 = 完成一個具體要求
  • 勾 L3 大項 = 完成大項（自動勾選所有 L4）
  • 勾 L2 段章 = 完成段章（自動勾選所有下層）
  • 勾 L1 獎章 = 完成獎章（自動勾選全部）

🔄 雙向級聯

勾選會自動聯動：

  • 勾上層 → 自動完成所有下層
  • 下層全完成 → 上層自動完成
  • 一天只能完成幾項？勾 L4 即可
  • 一次完成整個大項？直接勾 L3

👥 兩種身份

根據角色提供不同功能：

  • 領袖：勾選成員完成項目
  • 成員：查看自己的進度
  • 全團總覽：查看所有成員
  • 按項目查看：特定項目狀況

新旅團部署指南

只需 10 分鐘，3 步完成

您的旅團會擁有獨立的 Google Sheet 後端，資料完全由您掌控。

1

下載後端程式碼

點擊下方按鈕下載 Code.gs 檔案：

下載 Code.gs
2

建立 Google Sheet 並部署

2.1 開啟 Google Sheets，建立新試算表
2.2 命名為「深資童軍進度追蹤」，複製網址中的 Sheet ID
2.3 點擊「擴充功能」→「Apps Script」
2.4 刪除預設內容，貼上 Code.gs 的內容
2.5 修改第一行的 SHEET_ID 為您的 Sheet ID
2.6 儲存後執行 initializeSheets 函數（會自動生成 API Key）
2.7 點擊「部署」→「新增部署」→「網頁應用程式」，存取權設為「任何人」
2.8 複製部署後的網址

3

提交給系統管理員

把以下資訊交給系統管理員，即可完成登記：

  • 旅團編號（例：0082）
  • Apps Script 網址（以 /exec 結尾）
  • API Key（執行 initializeSheets 時顯示的隨機字串）

管理員會將您的旅團加入系統，之後您和成員就能直接使用。

4

（選用）接入主系統

如果您的旅團有使用 scoutsystem-2.0 主系統，可以把此系統接入：

  • 把上述 3 項資訊交給主系統管理員
  • 管理員會在主系統的「元件設定」中加入您的旅團
  • 設定完成後，從主系統 Dashboard 的元件卡片就能直接進入
  • 身份會自動帶入，不需要再選擇旅團和身份

提示：獨立使用和接入主系統可以同時使用，不衝突。 領袖可以從主系統進入（身份自動帶入），成員可以直接 bookmark 前端 URL。

資料安全說明

  • 資料存在您自己建立的 Google Sheet，完全由您掌控
  • API Key 由系統自動生成，防止爬虫存取
  • Google Sheet 需要登入才能編輯，保護您的資料
  • 忘記 API Key？在 Apps Script 執行 showApiKey() 即可重新顯示

📖 使用教學

1

選擇旅團

從旅團列表中選擇您的旅團，系統會自動載入旅團設定

2

選擇身份

填寫 YMIS（成員必填）

成員需要填寫自己的 YMIS（10位數字），領袖可以留空

4

開始使用

系統會自動載入您的進度，建議加入書籤方便下次使用

四級勾選說明

領袖勾選操作

  • 勾選 L4 小項：完成一個具體要求（例：第1次團活動）
  • 勾選 L3 大項：自動完成所有小項（例：完成露營全部要求）
  • 勾選 L2 段章：自動完成所有大項和小項（例：完成活動策劃段章）
  • 勾選 L1 獎章：自動完成該獎章所有下層項目

提示：所有下層完成後，上層會自動標記為完成。領袖只需勾選已完成的層級即可。

批量保存

勾選後資料會暫存在瀏覽器，點擊底部的「確認寫入」按鈕才會儲存到 Google Sheet。這樣可以一次性勾選多項，避免頻繁等待。
