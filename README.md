# 🔥 深資童軍進度性獎章追蹤系統 v2.0

> 基於 2026 年第 11 版《深資童軍訓練綱要》  
> 支援四級進度追蹤、級聯自動完成、批量確認寫入

---

## 這是什麼？

一個追蹤深資童軍進度性獎章的系統，從會員章到榮譽童軍獎章，四級進度一目了然。

```
L1 會員章 → L2 肩章 → L3 深資童軍獎章 → L4 榮譽童軍獎章
```

- ✅ 領袖可以勾選完成項目，自動記錄日期
- ✅ 勾上層自動完成所有下層（級聯）
- ✅ 全團總覽 + 按項目交叉查看
- ✅ 成員可以看到自己的進度
- ✅ 瀏覽器自動備份（防當機）

## 如何使用？

### 方式 1：從連結進入（推薦）

管理員會給你一個連結，例如：

```
https://vsbadge.vercel.app/?u=0082
```

直接打開就能用，身份自動帶入。建議加入書籤。

### 方式 2：獨立使用

直接打開：

```
https://vsbadge.vercel.app/
```

會顯示旅團選擇列表，選擇你的旅團後就能使用。

### 方式 3：接入主系統

如果你的旅團有使用 scoutsystem-2.0 主系統，可以從 Dashboard 的元件卡片進入，身份自動帶入，更方便。

## 旅團部署流程

### 旅團要做的事（10 分鐘）

1. 建立 Google Sheet
2. 貼上 Code.gs（系統管理者提供）
3. 執行 initializeSheets
4. 系統會自動生成 **API Key**
5. 部署 Apps Script
6. 把 **Apps Script URL + API Key** 交給系統管理員

### 系統管理員要做的事

收到旅團的資訊後，編輯 `troops.json`：

```json
{
  "troops": {
    "0082": {
      "name": "第 82 旅",
      "backend": "https://script.google.com/macros/s/XXXXXX/exec",
      "apikey": "vs_xxxxxxxxxxxxxxxxxxxx"
    }
  }
}
```

提交後 Vercel 自動部署，旅團就能使用了。

## 四級架構

| 級別 | 內容 | 項目數 |
|------|------|--------|
| L1 會員章 | 活動、理解、禮節、許諾、保護兒童 | ~18 |
| L2 肩章 | 認識、露營、繩結、遠足、急救 | ~24 |
| L3 獎章 | 活動策劃、社會服務、多元技能、戶外探險 | 70+ |
| L4 榮譽童軍 | 4 條金帶 | 20+ |

### 級聯自動完成

```
勾 ✓ 會員章      → 所有理解、禮節、許諾... 全部自動完成
勾 ✓ 露營        → 露營內 7 個子項全部自動完成
所有子項 ✓       → 露營自動標記完成
4 個段章全 ✓     → 深資童軍獎章自動完成
```

## 安全分層

| 層級 | 保護對象 | 機制 |
|------|---------|------|
| Google Sheet 密碼 | 擋人 | 只有 Sheet 擁有者能登入修改 |
| API Key | 擋 AI/爬虫 | 沒有 Key 就不能用 Apps Script API |
| troops.json | 系統管理員管理 | 旅團無法修改，只有管理員能更新 |

## 常見問題

**Q: 我該用獨立使用還是接入主系統？**  
A: 如果你的旅團有使用 scoutsystem-2.0，建議接入主系統（身份自動帶入更方便）。如果沒有，用獨立使用就好。

**Q: 兩種模式可以同時用嗎？**  
A: 可以。領袖從主系統進入，成員直接 bookmark 前端 URL。

**Q: 需要密碼嗎？**  
A: 不需要密碼。獨立使用時靠 API Key 保護，接入主系統時身份由主系統帶入。

**Q: 資料存在哪裡？**  
A: 存在旅團自己建立的 Google Sheet 裡，旅團完全擁有資料。

**Q: 忘記 API Key 怎麼辦？**  
A: 在 Apps Script 編輯器中，執行 `showApiKey()` 函數，會再次顯示 API Key。

**Q: 勾選後沒有保存？**  
A: 記得按底部的「確認寫入 Google Sheet」按鈕。勾選時資料暫存在瀏覽器，只有按確認才會寫入。

|------|------|
