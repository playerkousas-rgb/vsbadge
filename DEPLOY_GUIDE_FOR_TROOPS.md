# 📋 旅團部署指南（給領袖看的）

> 你只需要做 3 步，10 分鐘完成。

---

## 你需要準備

- ✅ 一個 Google 帳號
- ✅ 10 分鐘時間
- ✅ 本指南

---

## 第 1 步：建立 Google Sheet

1. 打開 [Google Sheets](https://sheets.google.com)
2. 建立新試算表
3. 取名為「深資童軍進度追蹤 - 你的旅團名稱」
4. 複製網址列中的 Sheet ID

   ```
   https://docs.google.com/spreadsheets/d/【這段就是 Sheet ID】/edit
   ```

---

## 第 2 步：貼上後端程式碼

1. 在剛建的 Sheet 中，點擊上方選單 **擴充功能** → **Apps Script**
2. 會打開一個新的 Apps Script 編輯器
3. **刪除**編輯器裡所有現有內容
4. 複製下方提供的 `Code.gs` 程式碼（聯絡系統管理者取得）
5. **全部貼上**到 Apps Script 編輯器
6. 修改最上面兩行：

```javascript
const SHEET_ID = '你的Sheet ID';    // ← 填入第1步複製的 ID
const API_KEY = '你自己設定的Key';    // ← 自己打一串（例如你的旅團名+隨機數字）
```

**API Key 建議格式：**
```
const API_KEY = 'troop0082_vs_badge_2026';
```
記住這組 Key，等一下要用。

7. 點擊上方 💾 **儲存** 按鈕
8. 在上方的函式選單中選擇 `initializeSheets`
9. 點擊 ▶ **執行**
10. 第一次會要求授權 → 選擇你的 Google 帳號 → 進階 → 前往 → 允許

完成後會看到 Sheet 裡自動新增了三張工作表：
- ✅ 成員名單
- ✅ 進度紀錄
- ✅ 項目定義

---

## 第 3 步：部署

1. 在 Apps Script 編輯器中，點擊左上角 **部署** → **新增部署**
2. 點擊 ⚙ 齒輪 → 選擇 **網頁應用程式**
3. 填寫設定：
   - **描述：** 深資童軍進度追蹤
   - **執行身分：** 我
   - **存取權：** 任何人
4. 點擊 **部署**
5. 複製產生的 **網頁應用程式網址**

   ```
   https://script.google.com/macros/s/XXXXXXXXX/exec
   ```

---

## ✅ 完成！如何使用？

### 獨立使用（不經主系統）

打開這個網址：

```
https://vs-badge-tracker.vercel.app/
```

（前端由系統管理者統一提供，所有旅團共用）

首次開啟會看到設定畫面，填入：

| 欄位 | 填什麼 |
|------|--------|
| 旅團編號 | 你的旅團編號（例：0082） |
| 我的身份 | 領袖 / 成員 |
| YMIS | 成員用，填自己的 YMIS |
| Apps Script 後端 URL | 第 3 步複製的網址 |
| API Key | 第 2 步你設定的 Key |

按「進入系統」→ 完成！設定會存入瀏覽器，下次自動帶入。

**建議：** 設定完成後，把網址加入書籤，下次直接從書籤進入。

### 接入主系統（經元件卡片）

聯絡系統管理者，提供以下資訊：

| 資訊 | 說明 |
|------|------|
| 旅團編號 | 例：0082 |
| 後端 URL | 第 3 步複製的 Apps Script 網址 |
| API Key | 第 2 步你設定的 Key |

系統管理者會在主系統幫你設定好，之後從 Dashboard 的元件卡片就能進入。

## 🆘 常見問題

**Q: 授權失敗？**
A: 確保你用建立 Sheet 的同一個 Google 帳號。Apps Script 首次執行時會要求授權。

**Q: 顯示「無法連接後端」？**
A: 確認 Apps Script 已部署為「網頁應用程式」且存取權設為「任何人」。

**Q: 忘記 API Key？**
A: 在 Apps Script 編輯器中打開 Code.gs，查看 `API_KEY` 那行。

**Q: 想新增/刪除成員？**
A: 進入系統後（領袖身份），在「進度」頁或「全團」頁按「+ 新增成員」。

**Q: 勾選後沒有保存？**
A: 記得按底部的「確認寫入 Google Sheet」按鈕。勾選時資料暫存在瀏覽器，只有按確認才會寫入 Google Sheet。如果中途關閉瀏覽器，下次開啟會提示恢復。

**Q: 多裝置同步？**
A: 只要有連接 Google Sheet 後端，所有裝置的資料都會同步。如果只是本地儲存模式（沒填後端 URL），則資料只在該瀏覽器。
